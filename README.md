Short presentation / reading for your code

“This is my HTML webpage project called Visit Durban.
The purpose of the page is to promote Durban as a tourist destination.

The webpage has three main sections:

Hero Section
This is the top part of the page. It welcomes visitors with the heading ‘Visit Durban Now’ and a short message about beaches, culture, and sports in Durban.
Activities Section
This section shows three popular activities in Durban:
relaxing at Durban Beach,
visiting uShaka Marine World,
and exploring Moses Mabhida Stadium.
Each activity has an image, a title, and a short description.
Tour Guide Section
This section introduces a guide who can help visitors explore Durban. It includes a guide image, a short introduction, and the guide’s name.

I used HTML to structure the content of the webpage, including headings, paragraphs, images, and sections.
The CSS file linked as style.css is used to style the page and make it look attractive and organized.”
